Simple Cron Jobs

  1. Create an operation class encapsulating your cron job.
  2. (You will not be using a batch class, just a single operation.)
  3. In a hook_cron implementation, pass an instance of the operation to CronCreateJobFromOperation.
  4. Set the maximum seconds you want to spend on the operation.
function my_module_cron() {
  $foo = new FooOperation('lorem', 'ipsum');
  (new CronCreateJobFromOperation($foo))
    ->setMaxTime(30)
    ->do();
}
